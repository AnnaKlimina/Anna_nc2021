(Информация взята на следующих сайтах: HTML и CSS)

Аннотация: данная страница показывает возможности форматирования и группировки текста html, а также основы использования CSS.

HTML — стандартизированный язык разметки документов для просмотра веб-страниц в браузере.

Важными тегами, используемыми для форматирования, являются теги для создания степеней и нижних индексов.
Первый тег - <sub> - нужен для создания нижних индексов. Подобное может пригодиться, например, для оформления логарифма:


log636

Второй тег - <sup> - нужен для создания степеней. Например:

e7 или (a + b)2 = a2 + 2ab + b2

Для выделения информации маркером используется тег <mark>.

На страницу сайта так же можно вставлять фрагменты програмного кода. Для выведения текста кода машинным шрифтом может использоваться тег <code> (форматирование не сохраняется). Например, вот CSS-код для оформления заголовка в начале сайта: h1{ text-align: center; text-decoration:underline; color: #135966; } Для выведения машинного кода с сохранением форматирования следует использовать тег <pre>. Вот тот же код с применением этого тега: 

        h1{
            text-align: center;
            text-decoration:underline;
            color: #135966;
          }

CSS — формальный язык описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки.
